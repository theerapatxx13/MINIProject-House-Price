# MINIProject-House-Price

## Objective
Predict house prices using Regression Models.

## Dataset
House Prices Dataset (~175,000 records)

## Models Used
- Linear Regression
- Ridge Regression
- Random Forest Regressor

## Results
- Linear R²: 0.48
- Ridge R²: 0.48
- Random Forest R²: 0.93
- Classification Accuracy: 89%

## Key Insight
Area is the most important feature (53% importance).
